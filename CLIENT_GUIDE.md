# Client Guide: How to Edit Your Website

## Accessing the Admin Panel

1. Go to your website's admin panel:
   ```
   https://YOURWEBSITE.netlify.app/admin/
   ```

2. Login with the credentials I provided

## How to Edit Content

### 1. Edit Text on Any Section

1. Click the section you want to edit (e.g., "Hero Section", "About", "Pricing")
2. You'll see form fields for each piece of content
3. Type your changes
4. Click **"Publish"** at the top right
5. Your site automatically updates!

### 2. Edit Pricing

1. Go to **"Pricing Section"**
2. Scroll to "Pricing Cards"
3. Click a card to edit:
   - Hours per week
   - Rate per hour
   - Monthly total
   - Featured (yes/no - shows as highlighted)
4. Click **"Publish"**

### 3. Update Contact Info

1. Go to **"Contact Section"** or **"Office Hours & Info"**
2. Edit address, phone numbers, hours
3. Click **"Publish"**

### 4. Add Images

1. Click **"Media"** in the top menu
2. Click **"Upload"** button
3. Select your image
4. Image is now available to use

## Common Tasks

### Change Prices
> Go to Pricing → Edit the numbers → Publish

### Add New Subject
> Go to Subjects → Add new card with title, grades, tags → Publish

### Update Office Hours  
> Go to Office Hours → Edit days/times → Publish

### Change Phone Number
> Go to Contact Section or Settings → Edit phone → Publish

## Important Notes

✅ **Do:**
- Edit text, prices, phone numbers, hours
- Upload images
- Add new pricing tiers or subjects

❌ **Don't:**
- Change site structure
- Delete sections
- Edit code or settings you don't understand
